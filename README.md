# OIBSIP_Web-Development-and-Design_Task-4
# Task 4: Login & Signup Authentication Page

## Objective  
Create a login and signup flow using JavaScript that validates user email and password, and upon successful login redirects the user to a secured page.

## Steps Performed  
- Built signup form to register user credentials (email & password) and login form for authentication.  
- Used JavaScript validation (email format, password length) and logic to check credentials.  
- Redirected users to a secured page after successful login.  

## Tools Used  
- HTML5  
- CSS3   
- JavaScript (form validation, redirect logic)
- Visual Studio Code

## Outcome  
A functional authentication page with signup and login capabilities, visual animation enhancements and redirect logic.
